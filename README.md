# pgwatch-ai
 AI copilot for PostgreSQL health monitoring
 ## Installation
pip install psycopg2 requests click

## Usage
python pgwatch_ai.py report
python pgwatch_ai.py ask "Why is my database slow?"

## Features
- Natural language database health queries
- Automated metrics collection
- AI-powered recommendations via Ollama (llama3)
- Fallback rule-based analysis when LLM unavailable
